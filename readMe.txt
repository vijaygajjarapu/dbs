Jungle Website has been automated using Cucumber with Page Object Model Framework
Scenarios have been placed in the feature files.
Page class contains elements and actions to act upon elements.
Browser config is a class file where Browser instantiation happens
Runner Class test is the execution start point where we provide tags.
Helper class provides re usable methods to be used in page class.

Covered for four different types of states whose tax percentage is different

How to execute :
1. It is a maven project,hence with proper maven setup in the local machine, navigate to pom.xml file location, open cmd and type 'mvn test' command
2. Import the project into eclipse as exiting maven project .
Open RunnerClassTest and click on Execute Button

Note: Chrome driver path has to be changed in the browserConfig file.
