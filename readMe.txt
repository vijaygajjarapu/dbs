Project is automated using Cucumber With Page Object Model
It Contains Feature files, step Definitions, page classes and An Extent Report to show all results.

How to execute :
1. It is a maven project,hence with proper maven setup in the local machine, navigate to pom.xml file location, open cmd and type 'mvn test' command
2. Import the project into eclipse as exiting maven project .
Open RunnerClassTest and click on Execute Button
